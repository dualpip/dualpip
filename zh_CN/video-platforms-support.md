## 外挂字幕格式支持

DualPiP 支持使用以下格式外挂字幕文件的任何网站：

**支持的格式：** VTT / SRT / ASS / EBUTT / TTML2 / XML / DFXP / JSON / TimedText

| 网站类型            | 支持范围     | 字幕支持 | 弹幕支持 |
| ------------------- | ------------ | -------- | -------- |
| 任意 HTML5 视频网站 | 通用解决方案 | ✅       | ❌       |

## 图例和说明

- ✅ **支持** - 功能可用且正常工作
- ❌ **不支持** - 该平台上功能不可用
- **聊天弹幕** - 实时聊天消息以弹幕叠加显示
- **缩略图预览** - 某些平台可能仅支持缩略图预览功能
- **通用播放器支持** - 适用于使用相应播放器技术的任何网站
- **外挂字幕支持** - 与任何支持外挂字幕文件的 HTML5 视频标签兼容
